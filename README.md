# Joyce 的个人网站

*Joyce's Portfolio — personal site & project showcase.*

这是 **Joyce** 的个人作品集站点：用一页清晰展示**我是谁、关注什么技术方向、有哪些个人项目与实践、工作经历**，以及如何联系我。页面**默认英文**，也可一键切换**中文**，方便不同读者阅读。

我目前在汇丰从事软件测试相关工作，同时具备 Java 后端与工程化实践背景；个人项目与练习会**持续在 [Gitee](https://gitee.com/joyceyu434840) 上更新**，网站上的展示也会随时间补充和调整。

## 网站上大致有什么

- **关于我**：背景、成长方向与当前关注点  
- **技能**：测试、Java / Spring 生态、数据与中间件等  
- **个人项目**：以卡片形式呈现代表性实践，**列表会随我的推送而变化**  
- **工作经历**：参与过的项目与职责摘要  
- **联系入口**：邮件与常用代码托管主页

## 技术与本仓库（简）

本站当前以 **HTML、CSS、JavaScript** 实现；个人项目侧常见技术包括 **Java、Spring Boot、Spring Cloud、MySQL、Redis、RabbitMQ** 等。我也会把本仓库当作前端练习的载体，**后续有计划逐步演进到 React + TypeScript** 等更贴近工程化的栈。

若你 fork 本仓库或参与维护：文案与项目列表在仓库内统一配置后生成页面，修改后本地执行 `npm run build`（即 `node build.js`）再提交即可；也可依赖 GitHub Actions 在推送后自动构建。
